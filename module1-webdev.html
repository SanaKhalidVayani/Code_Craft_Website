Module 1: HTML Basics

Introduction to HTML and understanding the structure of web pages

Lesson Video

Notes & Transcript

  • HTML stands for HyperText Markup Language
  • It is used to create the structure of web pages
  • HTML uses tags to define elements
  • Basic structure includes: <!DOCTYPE>, <html>, <head>, <body>
  • HTML files have a .html extension

Practice Task

Create a basic HTML page that displays your name and a short introduction.

Starter Code:

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>

</body>
</html>

Expected Output:

A webpage showing your name and a short paragraph about you.
