Module 4: Responsive Design & Bootstrap

Learn how to build mobile-friendly websites using Bootstrap 5

Lesson Video

Notes & Transcript

  • Responsive design makes websites work on all screen sizes
  • Bootstrap is a front-end framework for faster UI development
  • The Bootstrap grid system uses rows and columns
  • col-md-6 means half width on medium screens and above
  • Utility classes help with spacing, text, and alignment
  • Navbar, cards, and buttons are pre-built Bootstrap components
  • Mobile-first design starts with small screens first

Practice Task

Create a responsive page with a navbar and three cards using Bootstrap grid system.

Expected Output:

A mobile-friendly layout that stacks cards on small screens and aligns them in a row on larger screens.
