Module 2: Text, Links & Images

Learn how to structure content using headings, paragraphs, links, and images

Lesson Video

Notes & Transcript

  • <h1> to <h6> are used for headings
  • <p> is used for paragraphs
  • <a> tag is used to create hyperlinks
  • <img> tag is used to insert images
  • src attribute defines image path
  • href attribute defines link destination

Practice Task

Create a webpage with one heading, one paragraph, one image, and one clickable link.

Expected Output:

A webpage showing text content, an image, and a working hyperlink.
