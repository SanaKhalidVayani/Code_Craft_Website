Module 3: CSS Basics & Styling

Learn how to make your webpages beautiful using Cascading Style Sheets

Lesson Video

Notes & Transcript

  • CSS stands for Cascading Style Sheets
  • CSS controls layout, colors, fonts, and spacing
  • Inline CSS is written inside an HTML tag
  • Internal CSS is written inside a <style> tag
  • External CSS is written in a separate .css file
  • Selectors target elements to apply styles
  • Classes start with a dot (.) and IDs start with a hash (#)

Practice Task

Create a webpage with a heading, paragraph, and button. Style them using an external CSS file.

Expected Output:

A styled webpage with custom colors, spacing, and fonts.
